# ميزانتي - برنامج المحاسبة الشخصي

برنامج محاسبي متكامل لإدارة المصروفات الشخصية، الديون، الأهداف والاستثمارات.

## المميزات
- إدارة الإيرادات والمصروفات
- متابعة الديون والأقساط
- تحديد الأهداف المالية
- متابعة الاستثمارات
- تقارير ورسوم بيانية
- يعمل بدون إنترنت
- واجهة عربية (فصحى ولهجة مصرية)

## طريقة التشغيل
افتح ملف `index.html` في أي متصفح حديث.

## التوافق
يعمل على جميع المتصفحات الحديثة وأجهزة الجوال والكمبيوتر.
